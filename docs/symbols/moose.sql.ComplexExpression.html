moose.sql.ComplexExpression(op)
Represents a complex SQL expression, with a given operator and one or more attributes (which may also be ComplexExpressions, forming a tree). This is an abstract class that is not that useful by itself. The subclasses @link moose.sql.BooleanExpression}, moose.sql.NumericExpression and moose.sql.StringExpression should be used instead of this class directly.
Parameters:
{...} op
The operator and arguments for this object to the ones given.

Convert all args that are hashes or arrays of two element arrays to moose.sql.BooleanExpressions, other than the second arg for an IN/NOT IN operator.

Throws:
{moose.sql.ExpressionError}
if the operator doesn't allow boolean input and a boolean argument is given.
{moose.sql.ExpressionError}
if the wrong number of arguments for a given operator is used.
