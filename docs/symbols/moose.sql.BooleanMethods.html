Class moose.sql.BooleanMethods

Defines boolean/logical AND (&), OR (|) and NOT (~) operators that are defined on objects that can be used in a boolean context in SQL (moose.sql.LiteralString, and moose.sql.GenericExpression).
Defined in: sql.js.

Class Detail
moose.sql.BooleanMethods()

Example 1 : 

sql.a.and(sql.b) //=> "a" AND "b"
sql.a.or(sql.b) //=> "a" OR "b"
sql.a.not() //=> NOT "a"
