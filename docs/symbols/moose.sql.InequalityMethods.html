Class moose.sql.InequalityMethods

This mixin includes the inequality methods (>, <, >=, <=) that are defined on objects that can be used in a numeric or string context in SQL.
Defined in: sql.js.

Class Detail
moose.sql.InequalityMethods()

Example 1 : 

sql.a.gt("b")  //=> a > "b"
sql.a.lt("b")  //=> a > "b"
sql.a.gte("b") //=> a >= "b"
sql.a.lte("b") //=> a <= "b"
sql.a.eq("b") //=> a = "b"
